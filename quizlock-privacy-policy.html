QuizLock 개인정보처리방침

시행일: 2025년 1월 17일

QuizLock(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 관련 법규를 준수합니다. 본 개인정보처리방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

1. 수집하는 정보

1.1 자동 수집 정보

  • 앱 사용 시간 데이터: 스마트폰 사용 시간을 측정하기 위해 기기의 앱 사용 통계에 접근합니다. 이 정보는 기기에만 저장됩니다.

1.2 사용자 제공 정보

  • 설정 정보: 사용자가 설정한 사용 시간, 퀴즈 난이도, 연령대 등의 정보
  • 반성 사진: 잠금 화면에서 촬영되는 사진 (기기에만 저장, 외부 전송 없음)

1.3 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처, 메시지 내용

2. 정보 사용 목적

  • 스마트폰 사용 시간 측정 및 표시
  • 설정된 시간 초과 시 퀴즈 제공
  • 사용자 설정 저장 및 유지

3. 정보 저장 위치

  • 기기 내 저장: 모든 개인 데이터(사용 시간, 설정, 반성 사진)는 사용자의 기기에만 저장됩니다.
  • 서버 저장: 퀴즈 문제 데이터만 서버에서 불러옵니다. 사용자 개인정보는 서버에 저장되지 않습니다.

4. 제3자 서비스

4.1 광고 서비스 (Google AdMob)

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 기기 식별자
  • 광고 상호작용 데이터

아동 사용자의 경우, COPPA 정책에 따라 개인화된 광고가 제한됩니다.

자세한 내용: Google 개인정보처리방침

5. 데이터 보안

앱은 사용자 데이터를 보호하기 위해 다음과 같은 조치를 취합니다:

  • 모든 민감한 데이터는 기기에만 로컬 저장
  • 외부 서버로 개인정보 전송 없음
  • 암호화된 데이터 저장

6. 아동 개인정보 보호

앱은 아동 사용자를 대상으로 할 수 있으며, COPPA(아동 온라인 개인정보 보호법) 및 관련 법규를 준수합니다:

  • 아동의 개인 식별 정보를 수집하지 않습니다
  • 아동 대상 광고는 개인화되지 않습니다
  • 부모 또는 보호자가 앱 설정을 관리할 수 있습니다

7. 사용자 권리

  • 데이터 삭제: 앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다.
  • 설정 초기화: 앱 내 설정에서 데이터를 초기화할 수 있습니다.

8. 개인정보처리방침 변경

본 방침이 변경될 경우, 앱 내 공지 또는 본 페이지를 통해 안내드립니다. 중요한 변경사항은 시행 전 사전 고지합니다.
