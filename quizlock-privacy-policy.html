QuizLock 개인정보처리방침

시행일: 2025년 1월 17일 | 최종 수정일: 2025년 2월 21일

QuizLock(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 관련 법규를 준수합니다. 본 개인정보처리방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

1. 수집하는 정보

1.1 자동 수집 정보

  • 앱 사용 시간 데이터: 스마트폰 사용 시간을 측정하기 위해 기기의 앱 사용 통계에 접근합니다. 이 정보는 기기에만 저장됩니다.

1.2 사용자 제공 정보

  • 설정 정보: 사용자가 설정한 사용 시간, 퀴즈 난이도, 연령대 등의 정보

1.3 기술적 정보 및 앱 권한

앱은 핵심 기능을 제공하기 위해 다음과 같은 기기 권한을 사용합니다:

  • 사용 정보 접근 (PACKAGE_USAGE_STATS): 앱 사용 시간 측정
  • 다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW): 잠금 화면 및 퀴즈 오버레이 표시
  • 부팅 시 실행 (RECEIVE_BOOT_COMPLETED): 기기 재부팅 후 사용 시간 모니터링 자동 재시작
  • 포그라운드 서비스 (FOREGROUND_SERVICE): 백그라운드에서 사용 시간 모니터링 유지
  • 인터넷 (INTERNET): 퀴즈 데이터 로딩 및 광고 표시
  • 네트워크 상태 (ACCESS_NETWORK_STATE): 네트워크 연결 상태 확인

1.4 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • GPS 기반 위치 정보
  • 연락처, 메시지 내용

2. 정보 사용 목적

  • 스마트폰 사용 시간 측정 및 표시
  • 설정된 시간 초과 시 퀴즈 제공
  • 사용자 설정 저장 및 유지
  • 앱 성능 개선 및 안정성 유지

3. 정보 저장 위치

  • 기기 내 저장: 모든 개인 데이터(사용 시간, 설정 등)는 사용자의 기기에만 저장됩니다.
  • 서버 저장: 퀴즈 문제 데이터만 서버에서 불러옵니다. 사용자 개인정보는 서버에 저장되지 않습니다.

4. 제3자 서비스

4.1 광고 서비스 (Google AdMob)

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 광고 ID (Google Advertising ID)
  • 기기 식별자 및 기기 정보 (모델, OS 버전 등)
  • IP 주소 (대략적 위치 추정에 사용될 수 있음)
  • 광고 상호작용 데이터 (클릭, 노출 등)
  • 앱 활동 데이터

아동 사용자의 경우, COPPA 정책에 따라 개인화된 광고가 제한됩니다.

자세한 내용: Google 개인정보처리방침

Google의 광고 데이터 사용 방법: Google 광고 정책

5. 데이터 보안

앱은 사용자 데이터를 보호하기 위해 다음과 같은 조치를 취합니다:

  • 모든 민감한 데이터는 기기에만 로컬 저장
  • 외부 서버로 개인정보 전송 없음
  • 암호화된 데이터 저장

6. 데이터 보존 및 삭제

  • 기기 내 저장 데이터: 앱을 삭제하면 즉시 모든 데이터가 삭제됩니다.
  • 제3자 서비스(AdMob)에서 수집한 데이터: 해당 서비스의 보존 정책에 따릅니다. 자세한 내용은 Google 개인정보처리방침을 참고하세요.
  • 서버 측 데이터: 퀴즈 문제 요청 시 발생하는 네트워크 로그는 보안 및 서비스 안정성 목적으로 최대 90일간 보존 후 자동 삭제됩니다.

  • 부모 또는 보호자가 앱 설정을 관리할 수 있습니다

  • 데이터 삭제: 앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다.
  • 설정 초기화: 앱 내 설정에서 데이터를 초기화할 수 있습니다.
